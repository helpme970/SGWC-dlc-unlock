# SGWC-dlc-unlock
Script designed to unlock all DLCs for Sniper Ghost Warrior Contracts 1 and 2.

**⚠️ Compatibility Note:**
This script does not work properly with the GOG version.
For instructions on using the GOG version, please see [here](https://github.com/helpme970/SGWC-GOG-to-Steam).

## Usage Instructions
1. Download `unlock.py`.
2. Run the script using Python.
3. Enter the folder path where you want the DLCs extracted.
4. Enter the game number you are using (1 or 2).

## Functionality
This script automatically generates `.xml` files which inform the game that the respective DLCs have been unlocked.
